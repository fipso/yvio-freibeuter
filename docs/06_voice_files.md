## Voice Line Format Syntax

In `files_de.txt`, voice lines use special syntax:

| Symbol | Meaning |
|--------|---------|
| ?? | Language-specific file (de/en) |
| ? | Root sound file |
| \| | Sequence separator |
| %variable | Variable placeholder |
| ~z | Number placeholder |
| ~z1, ~z2 | Multiple number placeholders |
| #reference | Reference to another voice line |

### Example
```
quest_woman_pt_2=??p0458.wav|??p0453.wav|~z|??p0078.wav
```
This plays: p0458.wav, then p0453.wav, then a number (~z), then p0078.wav.

## Code References

- Voice line parsing: Uses `sub_2155C` and `sub_461C` functions
- Number announcement: Uses phX.wav (hundreds) + ptXhY.wav (thousands) system
- Officer selection: Based on player ID (1-4)
